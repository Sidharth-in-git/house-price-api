# House Price Prediction API

### This project is a RESTful API for predicting house prices based on key features like area, number of bedrooms, and bathrooms. It uses Python, FastAPI, and scikit-learn for machine learning, and provides an interface to get price predictions dynamically.

## Features

* Predict house prices using a trained machine learning model.

* Built with FastAPI for fast API development.

* Supports CORS for frontend integration.

* Easily extendable for more features (e.g., location, amenities).


## Tech Stack

* Backend: Python, FastAPI

* Machine Learning: scikit-learn, pandas

* Model Persistence: Pickle

* Frontend Integration: HTML/Bootstrap (optional)

* API Documentation: Swagger UI (auto-generated by FastAPI)

## Running the API

1.  Start the FastAPI server:
```uvicorn main:app --reload ```

2. Open your browser and access:

* API root: http://127.0.0.1:8000

* Swagger UI: http://127.0.0.1:8000/docs

* Redoc: http://127.0.0.1:8000/redoc

## API Endpoints
POST /predict

Predict house prices using JSON input:

Request Body:
```{
  "area": 1500,
  "bedrooms": 3,
  "bathrooms": 2
}
```
Response:
```
{
  "predicted_price": 250000
} 
```

## Frontend Integration

#### You can integrate this API with a Bootstrap frontend:
```<form id="predictForm">
  <input type="number" id="area" placeholder="Area in sq.ft">
  <input type="number" id="bedrooms" placeholder="Bedrooms">
  <input type="number" id="bathrooms" placeholder="Bathrooms">
  <button type="submit">Predict Price</button>
</form>

<script>
document.getElementById('predictForm').addEventListener('submit', async (e) => {
    e.preventDefault();
    const response = await fetch('http://127.0.0.1:8000/predict', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            area: parseInt(document.getElementById('area').value),
            bedrooms: parseInt(document.getElementById('bedrooms').value),
            bathrooms: parseInt(document.getElementById('bathrooms').value)
        })
    });
    const data = await response.json();
    alert(`Predicted Price: ${data.predicted_price}`);
});
</script>
```

## Training Your Own Model

2. Train a Linear Regression or any model of your choice.

3. Save the model using Pickle:
